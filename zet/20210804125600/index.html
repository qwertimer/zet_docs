Using recursive descent in bash

We can use the continue statement to break out of logic if true. This is useful if you are parsing some lines of input. For example you may want to first check if it contains a #. If it doesn't then we continue. If it does then we check the next character and follow that logic.

For more explanation about this checkout [https://www.youtube.com/watch?v=o_U76-7LmTQ] ~45mins.
